# Pi 文档

> 本页面是 [Pi 官方文档](https://pi.dev/docs/latest) 的中文翻译。仅供学习参考。

Pi 是一个极简的终端编码助手。它的核心设计保持轻量，同时通过 TypeScript 扩展、技能（Skills）、提示模板（Prompt Templates）、主题和 Pi 软件包进行扩展。

## 快速开始

通过 npm 安装 Pi：

```bash
npm install -g --ignore-scripts @earendil-works/pi-coding-agent
```

`--ignore-scripts` 在安装期间禁用依赖生命周期脚本。Pi 的正常 npm 安装不需要安装脚本。

在 Linux 或 macOS 上，也可以使用安装脚本：

```bash
curl -fsSL https://pi.dev/install.sh | sh
```

### 卸载

要卸载 Pi 本身，curl 和 npm 安装均使用 npm：

```bash
npm uninstall -g @earendil-works/pi-coding-agent
```

对于 pnpm、Yarn 或 Bun 安装，使用对应的全局删除命令：

```bash
pnpm remove -g @earendil-works/pi-coding-agent
```

```bash
yarn global remove @earendil-works/pi-coding-agent
```

```bash
bun uninstall -g @earendil-works/pi-coding-agent
```

卸载 Pi 后，设置、凭据、会话和已安装的 Pi 包会保留在 `~/.pi/agent/` 中。

然后在项目目录中运行：

```bash
pi
```

使用 `/login` 进行订阅提供商认证，或者设置 API Key（如 `ANTHROPIC_API_KEY`）后再启动 Pi。完整的新手流程请参阅 [Quickstart](/docs/latest/quickstart)。
